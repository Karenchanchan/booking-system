# Therapy Room Booking System A bilingual (English/Chinese) booking system for therapy rooms. ## Features - Client booking interface - Admin management panel - Room rate management - Monthly statistics and reporting - Bilingual support ## Setup 1. Clone repository 2. Install requirements: `pip install -r requirements.txt` 3. Run: `python app.py` 4. Access: http://localhost:5000 ## Default Accounts - Admin: admin / admin123 - Clients: Created by admin ## GitHub Sync To sync data across devices, ensure the database file is properly versioned or use a cloud database.
